# 🌍 UI/UX Pro Max: Diretrizes "Africa Dynamic"

**Foco:** Clareza, Funcionalidade, Vibração Local.
**Mood:** "Golden Hour na Savanna" (Luz, Quente, Acolhedor).

## 🎨 1. Paleta de Cores (Warm Earth & Clarity)
Abandonamos o Dark Mode total. Vamos para um **Light Mode Rico**.

*   **Background Base:** `#FDFBF7` (Off-White Papel / Areia Clara).
*   **Surface Cards:** `#FFFFFF` (Branco Puro) com sombra suave.
*   **Primary Action:** `#C05621` (Terracotta/Pôr do Sol) - *Botões, Links importantes.*
*   **Secondary Accent:** `#2F855A` (Kirstenbosch Green) - *Ícones de natureza, Sucesso.*
*   **Text Strong:** `#2D3748` (Slate Black) - *Leitura nítida.*
*   **Text Muted:** `#718096` (Cinza Azulado).
*   **Highlight:** `#F6E05E` (Sol Africano) - *Tags de destaque.*

## 📐 2. Tipografia & Hierarquia
*   **Headings:** Fonte Serifada com personalidade (ex: `Playfair Display` ou `Merriweather` via Google Fonts).
    *   *H1:* Grande, ousado, cor Terracotta.
*   **Body:** Sans-serif limpa (`Inter` ou `Outfit`).
*   **Legibilidade:** Contraste Alto. Nada de cinza claro sobre fundo branco.

## 🖼️ 3. Imagens & Dinamismo
*   **Hero Images:** Cada página deve ter um "Header Visual" (Banner) com foto de alta qualidade do local.
*   **Cards com Imagem:** Os cards de itens (Hotéis, Restaurantes) devem ter thumbnails visíveis.

## 🧩 4. Estrutura de Navegação (Funcionalidade)
O usuário quer navegar de 3 formas. O Dashboard deve oferecer esses 3 caminhos claros:

1.  **📅 Cronológico (Roteiro Mestre):**
    *   *Timeline Vertical.* Dia a dia.
    *   *Indicador visual:* "Onde estamos hoje?"

2.  **📍 Por Local (Destinos):**
    *   *Grid de Cards:* Cape Town, Kruger, Vic Falls, Winelands.
    *   *Ao clicar:* Abre filtro daquela cidade.

3.  **🏷️ Por Categoria (Interesses):**
    *   *Pílulas/Tags:* Hotéis, Restaurantes, Experiências, Logística.

## 🚀 5. Componentes UI Reutilizáveis
*   **"Passport Stamp" Tags:** Tags arredondadas que parecem carimbos.
*   **Map Markers:** Ícones de localização coloridos.
*   **Progress Bar:** Para checklist de tarefas pré-viagem.
